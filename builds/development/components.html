AXELERATOR COMPONENTS

Axelerator is a modular, flexible platform. It allows you to select the items you
need and configure them at a granular level to your exact requirements.

Axelerator| Benefits| Components

ENGINEERED FOR CHANGE

Axelerator is designed with insurance users in mind. It’s built by developers with intimate knowledge your industry, and a clear understanding that the requirements you seek are unique and ever-evolving.

With Axelerator, you aren’t held back by rigid systems, inflexible IT infrastructure and unproductive development tools – you’re in control.

Complete Insurance functionality with flexibility

Axelerator manages life insurance and general insurance policies and claims in a single platform. Unlike other hard coded systems, Axelerator’s core modules and components are shared across the entire system. Tailoring to suit your products and processes is mostly done through configuration and rules – not coding.

Sample working models

Axelerator helps you get a head start on new projects with sample configurations for policy and claims processing.

These initial working systems can be readily tailored to your requirements using the extensive rules and configuration options available.

Rules engine

At the heart of Axelerator is its powerful rules engine.

Unlike most other rules engines which manage only business logic, our rules engine drives the entire system including business logic, user interface, data validation, integration, process flow and calculations.

The engine is incredibly powerful. We are yet to identify any constraints to Axelerator’s capacity.

Dynamic and responsive forms

All forms (quotes, application forms, search panels, help and information panels, and tasks) are driven by rules. Each is dynamically generated and is responsive to information input without requiring the user to hit send.

Workflow functionality

Axelerator provides great workflow functionality that is easy to configure and highly flexible, so you can start making progress from the get-go.

Axelerator guides users through processes. It controls the sequence and provides system alerts when actions are overdue.

Rules Studio

The Axelerator Rules Studio provides a user-friendly drag and drop tool for defining and managing rules. Our simple and easy to navigate Studio tool requires no prior programming knowledge to use.

Desktop version

To help you work anywhere, Axelerator has a stand-alone module that runs on a PC or laptop, with or without internet.

Support for mobile phones and tablets

Axelerator’s user interface is based on the Bootstrap CSS to provide a responsive interface that adjusts to all mobile devices.

Document management

Axelerator can generate documents in Word, PDF and Excel, including graphs and charts.

User access and security

Axelerator enables full user control over user registration, authentication and authorisation. The user management module can run stand-alone or be integrated with your preferred system.

Integration services

Axelerator comes with inbuilt APIs and web services to integrate with other systems to send and retrieve information.

Enterprise class Java foundation

Axelerator is built on current Java technology. It works with the major JEE application servers including WebSphere, WebLogic and JBoss and leading databases such as DB2, Oracle, Sybase, SQL Server and PostgreSQL.

The current Java technology provides high levels of capability and productivity; the supported JEE platforms ensure performance, scalability and security.
